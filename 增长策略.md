# 堆柴 — 用户增长与留存策略

> 构思于 2026-05-16

---

## 一、核心命题

堆柴要解决两个问题：
1. **拉新** — 让更多运动爱好者知道并下载
2. **留存** — 让用户持续打开、发布、互动，不流失

围绕「柴火+运动+社交」的三角，以下是具体策略。

---

## 二、游戏化玩法（让用户上瘾）

### 2.1 签到打卡进阶

```
基础：每日签到得1根柴火
├── 🏅 连续7天 → 额外+5根 + 「毅力柴」勋章
├── 🏅 连续30天 → 额外+30根 + 「老柴火」限定勋章
├── 🏅 连续100天 → 额外+100根 + 「千日柴」特殊徽章
└── 🏅 连续365天 → 终身成就称号「火炬大使」
```

**设计点：** 断签不归零，可用柴火补签（3根补一天），减少挫败感。

### 2.2 场地探索任务

| 任务 | 奖励 |
|------|------|
| 打卡3个不同场地 | 20根柴火 |
| 打卡10个不同场地 | 100根柴火 + 「场地猎人」勋章 |
| 打卡本区全部场地 | 荣誉称号「区霸」 |
| 发布带视频的场地 | 额外+30根柴火 |
| 周五/周六发布 | 双倍柴火（周末效应） |

### 2.3 柴火燃烧挑战

每月设置「燃烧挑战」：
- 本月累计添柴满100根 → 解锁限定头像框
- 本月累计添柴满1000根 → 全服通报 + 特殊称号
- 柴火消耗本身就有排名，让用户为了排名而添柴

### 2.4 开盲盒/宝箱

- 每次发布场地 → 随机开一个「柴火盲盒」（1-200根随机）
- 每次被添柴 → 有一定几率获得「幸运回火」（系统奖励的1根）
- 首次登录每天转动一次「柴火轮盘」

## 三、社交裂变（让用户拉用户）

### 3.1 邀请机制升级

```
基础：邀请好友得10根柴火

升级玩法：
├── 📱 邀请排行榜 — 每月邀请TOP10全服展示 + 限定称号
├── 👫 邀请组队 — 和好友一起邀请，累计达标双方额外奖励
├── 📎 场地邀请码 — 分享场地时带你的邀请码，别人通过场地页注册也算你的
└── 🎁 邀请回流 — 被邀请人7天内活跃，邀请人额外得柴火
```

### 3.2 俱乐部裂变

- 俱乐部可生成专属邀请海报（带二维码+俱乐部头像+标语）
- 创建俱乐部即得50根柴火启动资金
- 俱乐部满10人/50人/100人 → 创建者获得对应勋章
- 俱乐部可发起「拉新擂台」— 哪个俱乐部周增成员最多，全俱乐部发福利

### 3.3 场地分享链

- 分享场地到微信 → 点进来的人看到「来自XX的推荐」
- 通过分享链接注册 → 分享者 + 被分享者各得5根
- 场地分享页面自带热度数据（当前柴火数+人气），让人有点进去的欲望

## 四、内容驱动（让用户有得看）

### 4.1 热力图/活动地图

- 首页地图模式显示「当前热度」— 用暖色到冷色表示附近场地的活跃度
- 红色=很多人去过/添柴多 → 吸引用户去探索
- 类似「大众点评必吃榜」但做运动版

### 4.2 运动日记/Timeline

- 用户的运动足迹地图（你去过哪些场地）
- 每次评论/添柴/发布 → 自动生成一条Timeline
- 年底生成「年度运动报告」：你去了几个场地、添了多少柴、走了多少步
- 可分享到朋友圈（刷屏级传播）

### 4.3 高质量内容推荐

- 精选场地推荐（编辑推荐/算法推荐）
- 「本周最火场地」「新晋场地」「冷门宝藏场地」
- 视频优先 — 运动场地的短视频比图片有冲击力得多
- 类似小红书的双列瀑布流 + 视频自动播放

### 4.4 运动百科/UGC指南

- 用户可写场地攻略/运动技巧（图文）
- 例如：「北京10大免费篮球场」「新手羽毛球拍推荐」
- 优质攻略可被推上首页，获得柴火打赏
- 类似「毒App」的社区内容 + 知乎的回答形式

## 五、赛事活动（让用户有期待）

### 5.1 每月主题活动

| 月份 | 主题 | 玩法 |
|:----:|------|------|
| 1月 | 新年第一跑 | 发布跑道场地双倍柴火 |
| 4月 | 春游运动季 | 户外场地专题 |
| 6月 | 毕业季约球 | 校园场地专题 |
| 7月 | 夏日水上运动 | 游泳/水上运动专题 |
| 10月 | 秋高气爽 | 登山/户外专题 |
| 12月 | 年度盛典 | 全年场地/俱乐部/用户评选 |

### 5.2 签到节日日历

- 重要运动节日自动触发活动（奥运会、世界杯、马拉松日）
- 当天发布相关运动场地 → 限定勋章

### 5.3 俱乐部联赛

- 月/季/年度俱乐部排行榜
- TOP10俱乐部获得虚拟奖杯 + 全站展示
- 俱乐部之间可发起「约战」— 在某场地比拼添柴数

## 六、本地化运营（让用户有归属感）

### 6.1 城市/区域分站

- 每个城市有独立首页，展示本地热门场地
- 城市榜单 + 城市俱乐部排行
- 城市站长（早期种子用户）可申请成为城市管理员

### 6.2 附近的人/附近的活动

- 地图上显示附近有哪些俱乐部正在活跃
- 附近正在进行的约球/约赛（Phase 2/3）

### 6.3 线下联动

- 和场地馆主合作，线下贴码 → 扫码进入场地详情 → 添柴送矿泉水
- 和运动品牌合作（李宁/安踏/迪卡侬）— 在热门场地做快闪活动

## 七、数据驱动增长

### 7.1 核心指标

| 指标 | 说明 |
|------|------|
| DAU/MAU | 日活/月活 |
| 用户人均发布场地数 | 内容生产力 |
| 柴火流通率 | 柴火池周转速度 |
| 邀请转化率 | 每个邀请带来多少注册 |
| 7日/30日留存 | 用户是否留下来 |

### 7.2 AB测试文化

- 签到奖励多少合适？（1根 vs 3根 vs 随机）
- 发布奖励多少合适？（50根 vs 100根 vs 200根）
- 邀请奖励多少合适？（5根 vs 10根 vs 20根）

---

## 八、总结：堆柴的增长飞轮

```
用户发布场地
  → 获得柴火（正向激励）
  → 场地吸引他人添柴/评论
  → 场地热度上升 → 上榜
  → 更多人看到 → 注册/下载
  → 新用户发布更多场地
  └── 循环 🔄
```

**俱乐部维度的飞轮：**
```
用户加入俱乐部
  → 参与活动 → 贡献柴火
  → 俱乐部排名上升
  → 吸引更多人加入
  └── 循环 🔄
```
